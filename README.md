# PMHScribe Trial List Cleanup

> **Tools:** *Microsoft Excel, SQL, Python (pandas)*

- Completed for PMHScribe, this project cleans and validates subscriber data in Python and SQL.
- Using emails from non-converted free trials and the active subscriber base, it removes records that exist in both and exports a clean Excel file to guide customer outreach.
- Cleaned data is used by PMHScribe for marketing campaigns and to reacquire lost subscribers.
